**Locked numeric constants (implement exactly):**
- Default idle grace: `300000` ms (5 minutes). Env override read in **daemon only**: `AGENT_TALKIE_RELAY_IDLE_MS` — when set to a non-empty string, parse as base-10 integer; if invalid or `< 0`, fall back to `300000`.
- Shutdown hard cap after SIGINT/SIGTERM: `10000` ms — after this, `process.exit(1)` even if `close()` has not settled.
- WebSocket “polite close” phase before `terminate()`: `2000` ms — attempt `client.close(1001, 'relay_shutdown')` for each open client, then wait up to 2000 ms total (e.g. `setTimeout` + counting `close` events), then call `terminate()` on any still open.

**RELAY-07 interpretation (per 03-RESEARCH.md A1):** “No pending protocol state” means **no open WebSocket connections** and **no idle-shutdown timer currently cleared by new activity** — not “SQLite rows must be empty.” SQLite durable memberships/session rows do **not** block idle shutdown.

**Health URL (exact):** `GET http://127.0.0.1:{port}/__agent-talkie/v1/health?generation={token}` where `{token}` is URL-encoded `relayGenerationToken`. Response `200` with body **exact JSON** (no extra keys): `{"ok":true,"generation":"<same token>"}`. Wrong or missing `generation` query → `403` with body `{"ok":false}`. Wrong HTTP method → `405`.

**`createRelayServer` new optional fields:**
- `relayGenerationToken?: string` — when defined, install the health handler on the shared `http.Server` **before** `server.listen`.
- `idleShutdownMs?: number` — when defined and **finite** and `>= 0`, enable idle shutdown (use `0` for tests only; production daemon passes parsed env or `300000`).
- `onIdleShutdown?: () => void | Promise<void>` — invoked once when idle timer fires; daemon implementation will `await` relay `close()`, `unlink` lockfile, `process.exit(0)`.

**Daemon IPC ready message (child → parent, exact shape):** serialize with `process.send` when `import.meta.url` is the daemon module and `process.send` is a function:
```json
{ "type": "relay.ready", "v": 1, "port": <number>, "generation": "<hex string>", "pid": <number> }
```
`generation` = exactly `randomBytes(16).toString("hex")` (32 lowercase hex chars). `port` = actual bound port from `createRelayServer` result. `pid` = `process.pid`.

**Daemon lockfile path:** `join(dataDir, "relay.lock")` where `dataDir` = `process.env.AGENT_TALKIE_DATA_DIR` if non-empty trimmed string, else `envPaths("agent-talkie", { suffix: "" }).data` from `env-paths@4.0.0`. Lock JSON on disk (pretty-print optional, keys required): `{"pid":<number>,"port":<number>,"generation":"<hex>"}`.

**Dependencies:** Add runtime deps to `packages/relay/package.json`: `"env-paths": "4.0.0"`.
